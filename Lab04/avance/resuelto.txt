Concluciones
 1. Si queremos agregar un label, input, etc, tenemos que importar dicho elemento para poder usarlo
 2. Se puede poner varios componentes y cada uno de ellos trabajara de forma independiente a la otra
 3. Para que funcione cada boton debe tener un handler la cual estara su accion a realizar
 4. Para modificar bien un componente, se tiene que saber el ciclo de vida de este, ya sea para agregar,
    actualizar o eliminar
 5. Se puede poner una condicional antes de que se ejecute los componentes para ver si los datos son correctos
